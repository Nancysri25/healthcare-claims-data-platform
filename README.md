# healthcare-claims-data-platform
End-to-end Healthcare Claims Data Platform built using Python, Apache Airflow, PostgreSQL, AWS S3, Docker, and Power BI.


## Overview

A production-style end-to-end data engineering project that processes US healthcare insurance claims using modern data engineering tools.

The pipeline ingests raw healthcare claims, validates and transforms the data, stores it in a PostgreSQL data warehouse, and provides analytics through Power BI dashboards.
